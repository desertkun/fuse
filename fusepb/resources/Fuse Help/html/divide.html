DivIDE Emulation

The DivIDE is another IDE interface for the Spectrum, of which full details can be found here. The interface can be activated via the DivIDE interface option on the Peripherals preferences dialog, and the state of its write protect jumper controlled via the DivIDE write protect option. If you're going to be using the DivIDE, you'll probably want one of the firmwares available from the DivIDE homepage.
